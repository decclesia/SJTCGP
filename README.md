# SJTCGP Card Database

Static GitHub Pages database and local deckbuilder for SJTCGP.

## July 2026 EX5 Pink Revolutionary Girl Utena update

- 626 cards with matching images and structured card text.
- Added EX5 Pink Revolutionary Girl Utena: EX5-001–021 and PUP-061–062.
- EX5-021 is a landscape JUMP Action with no Traits.
- EX5-019, EX5-020, PUP-061 and PUP-062 are JUMP Units with no Counter field.
- The Utena Guard Token and single SJ Marker are installed in Arena and linked to EX5-001.
- TCG Arena uses a separate counter-rotated copy for EX5-021 so its horizontal renderer displays the printed face upright on the field.
- Click any database or deckbuilder card to open its full stats, Traits, and effect.
- Keywords and timing windows use the project's green, purple, orange, and red text system.
- Search covers names, effects, Traits, stats, card numbers, releases, sets, colors, and card categories.
- Filter state is reflected in the URL, so planner links can open exact Release, Set, Color, Type, search, or individual-card views.
- Existing filters, alternate artwork controls, deck validation, local saving, and deck-image export remain available.

Upload every file and folder in this package to the GitHub Pages repository root, replacing files with the same names. Keep `card-text-data.js` and `card-text.json` beside `index.html`.
